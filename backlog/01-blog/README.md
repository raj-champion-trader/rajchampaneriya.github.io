# Blog Content

This is your core knowledge engine. Each subdirectory represents a content pillar.

## Content Categories

### technical-concepts/
Deep dives into architecture, system design, and cloud patterns.
- Reference architectures
- Trade-off analysis
- Technology explainers

### point-of-view/
Your architectural opinions and perspectives.
- Position papers
- Industry commentary
- Thoughtful takes on trends

### experience-learnings/
Lessons from real delivery work.
- What worked, what didn't
- Project patterns
- Reflections on complex problems

### open-letters/
Strategic viewpoints and leadership thinking.
- Open letters to the community
- Longer-form strategic thinking
- Advice based on experience

### book-reviews/
Book insights mapped to real work.
- Technical book summaries
- Practical applications
- Key takeaways

## Naming Convention

Use format: `YYYY-MM-DD-title-slug.md`

Example: `2026-01-19-architecture-decision-framework.md`

See `/07-automation/templates/blog-post-template.md`

---

**Target**: 1 post per week, rotating categories

**Style**: Clear, practical, experience-based
