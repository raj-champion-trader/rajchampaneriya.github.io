# Video Production Hub

Central location for all video creation assets and workflow.

## Workflow

```
scripts/ → audio/ → diagrams/ → published/
```

1. Write script
2. Record audio
3. Create diagrams
4. Edit in CapCut
5. Export and publish

## Folder Structure

### scripts/
- Written scripts for each video
- Format: `script-topic-name.md`
- Template: see `/07-automation/templates/video-script-template.md`

### audio/
- Raw audio recordings (.wav preferred)
- Format: `audio-topic-name.wav`
- Tool: Audacity

### diagrams/
- Excalidraw diagrams (.excalidraw files)
- Exported SVGs for video
- Format: `diagram-topic-name.excalidraw`

### published/
- Final video files
- YouTube links
- Publishing metadata
- Format: `YYYY-MM-DD-topic-name.md` (contains YouTube link, description, etc.)

## Video Types

### Quick Explainers (60-90 seconds)
- One concept, clearly explained
- Diagram-driven
- Calm, confident voice

### Architecture Walkthroughs (3-5 minutes)
- System design breakdown
- Trade-off discussion
- Visual + narration

### Thought Leadership (2-3 minutes)
- POV or insight
- Less technical, more strategic
- Text + minimal visuals

## Production Standards

- **Audio**: Clear, minimal editing required
- **Visuals**: Simple diagrams, professional look
- **Pacing**: Conversational, not rushed
- **Length**: As short as possible while being complete

## Tools

- **Audio**: Audacity
- **Diagrams**: Excalidraw
- **Video Editing**: CapCut
- **Microphone**: TBD
