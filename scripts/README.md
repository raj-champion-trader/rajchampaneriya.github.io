### deploy.sh
Interactive deployment script for publishing your site to GitHub Pages.

**Usage:**
```bash
./deploy.sh
```

**What it does:**
- Checks for git remote configuration
- Verifies you're on the main branch
- Stages and commits changes
- Pushes to GitHub
- Provides helpful prompts throughout

**First-time setup:**
The script will guide you through setting up your GitHub repository if it's not already configured.
