# Automation - Productivity Edge

Templates, checklists, and scripts to make content creation sustainable.

## Purpose

This folder contains everything that makes the content system:
- Repeatable
- Low-friction
- Consistent
- Sustainable

## Structure

### templates/
Reusable templates for different content types.

Files to create:
- `blog-post-template.md`
- `video-script-template.md`
- `architecture-kata-template.md`
- `event-documentation-template.md`
- `whitepaper-summary-template.md`

### publishing-checklists/
Step-by-step checklists for publishing workflow.

Files to create:
- `blog-publishing-checklist.md`
- `video-publishing-checklist.md`
- `weekly-content-checklist.md`

### scripts/
Automation scripts (future).

Potential scripts:
- Generate blog post frontmatter
- Create new kata folder structure
- Convert markdown to Hugo format
- Social media post generation

## Philosophy

**Start Manual, Automate When Painful**

Don't build automation until you've done the manual process 5-10 times and understand:
- What actually takes time
- What's error-prone
- What's repetitive

Then automate the pain points.

## Current Priority

Phase 1: **Templates only**  
Phase 2: **Checklists**  
Phase 3: **Scripts** (when needed)

---

**Value**: Removes friction, ensures consistency, prevents burnout  
**Maintenance**: Update templates based on what works  
**Future**: GitHub Actions for deployment automation
